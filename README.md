# Project

## Instructions:
1. Clone the repository or pull the repository
   ```
   git init
   git remote add origin https://github.com/Group4CPAN212/Project.git
   git branch -m master main # make sure we're on main, not master
   git pull origin main
   ```
3. Run ```npm install``` to install all the packages
4. run ```npm start``` to run the project

## Website
Once on the website, you are able to register. You can register, then log in.
You are able to browse movies, and create your own.
You are allowed to edit or delete your own movies.
You can not edit or delete other user's movies, but you are able to view them.
